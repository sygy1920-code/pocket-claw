# Pocket Claw - 启动说明

## ⚠️ 重要：必须使用正确的命令

### ✅ 正确的启动方式
```bash
cd /Users/yge/Project/pocket-claw
npm run dev
```

### ❌ 错误的命令（会导致问题）
```bash
vite                    # ❌ 不要直接运行 vite
npm run                 # ❌ 不要运行其他脚本
npx vite                # ❌ 不要使用全局 vite
```

## 🔧 如果遇到问题

### 1. 清理所有进程
```bash
pkill -f "electron"
pkill -f "vite"
```

### 2. 清理缓存和重新安装
```bash
rm -rf node_modules package-lock.json .vite dist-electron
npm install
```

### 3. 重新构建
```bash
npm run build
```

### 4. 启动开发模式
```bash
npm run dev
```

## 📁 验证文件结构

运行前确认以下文件存在：
- ✓ `out/main/index.js` (主进程)
- ✓ `out/preload/index.js` (预加载脚本)
- ✓ `out/renderer/index.html` (渲染进程)

如果有 `dist-electron/` 目录，说明配置错误，需要删除。

## 🐛 调试

如果仍有问题，运行诊断：
```bash
./diagnose.sh
```
